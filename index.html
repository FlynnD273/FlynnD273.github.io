About Me

I'm a CS Major at Worcester Polytechnic Institute, and was previously a homeschooled part-time Running Start student. I have an avid interest in mechanical engineering and computer programming.

I have competed in FIRST Tech Challenge (FTC). In the first year, I learned how to effectively use the CAD package my team was using, Fusion 360. I designed components to be 3D printed, and learned how to model the components to eliminate the need for print supports while keeping the components strong.

In the following years, I took on the role of hardware lead, and eventually team captain. I was in charge of keeping everything on schedule, and I was the major contributor to the overall robot design in CAD. I also manufactured many of the custom parts we designed, and worked with my teammates to make sure everything ran smoothly. As team captain, I worked closely with the hardware and software leads to make sure everything was progressing quickly and efficiently.

For software, I have been programming from a very young age. I started with MIT's Scratch when I was about 8, and haven't stopped programming since. After outgrowing Scratch's limitations, I moved on to learn JavaScript, C# for Windows Presentation Framework and Unity3D, as well as Java for Android development and for FTC. More recently I've been learning Python, since a lot of tools that I'm interested in are easily accessible through that. Using Python, I've been learning how to use libraries such as NumPy and OpenCV.
